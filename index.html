Tentang Saya

Halo gaes.....nama saya Muhammad iqbalhadi, lahir di Jakarta 7 Juli 2004,saya tinggal di Negara Slipi tepatnya disebrang Petamburan VII.Saya suka mendengar dan main musik, dan saya terkadang juga suka menonton band di Gigs kecil sendiri atau dengan teman saya. Selain itu saya juga suka belajar hal baru yang membuat saya penasaran, saya juga suka mengoleksi CD musik, seperti: The upstairs, Grrrl gang, dan yang baru-baru ini bada CD Jimi Jazz.

Saya sekolah di SMKN 19 Jakarta, sebelum saya SMK saya pernah sekolah di SDN 01 pagi Slipi tepatnya di sebrang Brimob, SMP saya di SMPN 61 Jakarta Barat. Selama saya bersekolah saya selalu senang sampai akhirnya ada virus tidak jelas yang melanda. Awalnya saya sempat senang belajar dirumah karena saya bisa bangun agak siang, tetapi pada akhirnya saya merasa bosan dirumah dan sangat ingin kembali sekolah seperti semula dan bertemu teman-teman disekolah. Saya sangat berharap bisa sekolah seperti semestinya saya juga yakin kalian juga ingin bermain atau berkumpul dengam teman dan saudara dengan normal, kita sebagai manusia hanya bisa berdoa dan menuruti pemerintah.

Barakah Always :)
